# 수익화 컨텐츠 마스터즈

고객관점 브랜드의 헤어디자이너 대상 인스타그램 마케팅 도구.

## 구성

- `index.html` — 앱 전체 (단일 HTML, 별도 빌드 불필요)

## 사용법

1. `index.html` 파일을 브라우저에서 열거나
2. GitHub Pages에 올려 URL로 접속

## 첫 실행

1. 사이트 접속 → 로그인 화면
2. 우측 상단 ⚙ 클릭
3. 관리자 비밀번호 입력
4. 새 코드 생성 (본인 이름 + 평생)
5. 발급된 코드 복사
6. 뒤로 → 코드 입력 → 비밀번호 설정 → 시작

## 기술

- 단일 HTML 파일, 외부 의존 없음
- 데이터는 브라우저의 localStorage에 저장
- 다른 PC/모바일에서는 데이터가 따라오지 않음 (추후 Supabase 도입 예정)

## 라이선스

비공개. 무단 복제·배포 금지.

## 앱처럼 설치하기 (PWA)

GitHub Pages URL을 모바일/PC 브라우저에서 열면 앱처럼 설치 가능해요.

### iOS (사파리)
1. 사이트 열기
2. 하단 공유 버튼 (□↑) 탭
3. "홈 화면에 추가" 선택
4. 홈에 "컨텐츠 마스터즈" 아이콘 생성

### Android (크롬)
1. 사이트 열기
2. 우측 상단 메뉴 (⋮) 탭
3. "앱 설치" 또는 "홈 화면에 추가"
4. 확인

### PC (크롬/엣지)
1. 사이트 열기
2. 주소창 우측의 설치 아이콘 (⊕) 클릭
3. "설치" 확인
4. 바탕화면에 아이콘 생성

설치된 앱:
- 브라우저 UI 없이 풀스크린
- 오프라인에서도 작동
- 이름: "컨텐츠 마스터즈"
- 아이콘: 트로피 (에메랄드)
